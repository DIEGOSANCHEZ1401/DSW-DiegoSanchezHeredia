Proposito de la pagina

Dar a conocer las diferentes formas, y los multiples usos del diseño del sitio web utilizando Frameworks CSS y hojas de estilo en cascada C

Hojas de Estilo en Cascada CSS,

CSS significa hojas de estilo en cascada (cascading style sheets), y es muy importante entender la palabra cascada. La forma en que se comporta la cascada es la clave para comprender el CSS. En algún momento trabajarás en un proyecto y encontrarás que el CSS que pensabas que debería aplicarse a un elemento no funciona. tomado de Pilamunga Poveda, E. M., & DT-López Sevilla, G. M. (2012). El maquetado a base de scripts y hojas de estilo en cascada (CSS) y su incidencia en la optimización de un sitio web.

Frameworks CSS

Es un conjunto de componentes para interfaz de usuario que te permite desarrollar aplicaciones basadas en HTML de manera rápida y efectiva. Todos los componentes son compatibles en móviles y diversos navegadores web tomado de Aponte, Á. M. V. (2014). Guía comparativa de Frameworks para los lenguajes HTML 5, CSS y JavaScript para el desarrollo de aplicaciones Web (Doctoral dissertation, Universidad Tecnológica de Pereira. Facultad de Ingenierías Eléctrica, Electrónica, Física y Ciencias de la Computación. Ingeniería de Sistemas y Computación).
